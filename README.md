# RD Compose

> Разработка заморожена на неопределённый срок.

# Используемые технологии:

* Модульный проект. Single activity. 🔠
* Like TEA подход к организации UI слоя. 🔭🌌🌠🪐✨
* Чистая архитектура. 🙈
* Jetpack Compose. 🤖
* Coroutines Flow. ♨️
* Firebase Firestore. 🔥
* Dagger Hilt. 🗡️
* Room, Compose DataStore. 💽
* Compose Navigation. 🗺️
